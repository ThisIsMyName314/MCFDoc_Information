### MCFDoc_Information

MCF Doc is a more basic version of JavaDoc which is designed to help
document datapacks thoroughly.

<br>
<br>

To create an MCFDoc valid function add a comment at the top of the .mcfunction file.

#### give_example.mcfunction

~~~
# Gives the executor the specified item

$give @s $(item) $(count)
~~~

This results in a single comment
